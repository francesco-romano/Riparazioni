Adding Data

In the main window you can add table entries. These data are automatically stored to the database when quitting the application, or by pressing File > Save or Command + S

Add data:

Press the Add button in the main window

A new row will appear in the table. You can insert data directly in the table, or in the text fields in the Details Panel (you can open that panel using the proper button in the top left of the main window). These data are compulsory:

■

Customer name.

■

Date.

■

Description.

■

Position.

If you need to delete some entries, select them, than press the Remove button.
