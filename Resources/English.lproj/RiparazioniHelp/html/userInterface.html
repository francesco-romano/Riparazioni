About the User Interface

You can choose between some printing options. They are explained in this section .

You can choose between:

■

Print Columns Headers.

If this option is checked, headers will be printed.

■

Print current view.

If this option is selected, you will print only the content of the table of the main window (included the same ordering)

■

Print all data.

If this option is selected, you will print all data, even if the table is filtered. Data will be sorted by Customer and Date.
