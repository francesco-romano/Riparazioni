Aggiungere una posizione

A position represent the place where you store some objects. It's a compulsory field if you want to add data to the database.

Adding a Position:

Press "Show Position". The Positions Panel will open.

In the Panel, press the Add button. A new row will appear in the table.

Type the position's name directly in the table or in the proper field.

In the position table, near the name of the position, there is a number. It represents the number of objects stored in that place.
