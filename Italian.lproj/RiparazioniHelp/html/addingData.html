Aggiungere dati

In the main window you can add table entries. These data are automatically stored to the database when quitting the application, or by pressing File > Save or Command + S

Aggiungi dati:

Premi il bottone "Aggiungi" nella finestra principale del programma.

Una nuova riga verrà inserita in fondo alla tabella. Puoi inserire i dati direttamente nella tabella, oppure negli appositi campi presenti nel pannello Dettagli (puoi aprire questo panello usando l'apposito bottone in alto a sinistra).
I seguenti dati sono obbligatori:

■

Cliente.

■

Data.

■

Descrizione.

■

Posizione.

Per eliminare delle righe, selezionale e poi premi il pulsante "Rimuovi".
